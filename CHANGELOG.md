# 更新日志 | Changelog

## [1.2.0] - 2026-01-15

### 新增
- ✨ **连续上涨阶段检测** - 新算法识别上涨趋势，允许5%以内回撤
- ✨ **上涨阶段可视化** - 图表绿色区域标注连续上涨阶段
- ✨ **斜率标注** - 显示上涨速度（%/天），加速事件显示🚀图标
- ✨ **图表放大功能** - 点击图表全屏查看，ESC关闭
- ✨ **一键启动脚本** - `./start.sh` 同时启动前后端

### API
- 🆕 `/api/uptrend_phases/{code}` - 获取连续上涨阶段
- 参数：`max_drawdown`(5%), `min_gain`(10%), `min_duration`(5天)

### 文档
- 📝 `docs/surge_model_design.md` - 急涨模型设计文档
- 📝 `docs/validation_framework.md` - 验证测试框架
- 📝 `docs/worklog.md` - 工作日志

## [1.1.0] - 2026-01-14

### 新增
- ✨ **复权净值计算** - 考虑分红再投资，准确显示真实收益率
- ✨ **新布局** - 一行 3 个基金的网格布局
- ✨ **时间区间选择器** - 支持 1年/3年/5年 统一切换
- ✨ **基金排序功能** - 按最新/名称/代码排序
- ✨ **基金删除功能** - 支持删除基金及其所有历史数据
- ✨ **固定顶部 Header** - 滚动时始终可见

### 优化
- 🔧 Tooltip 智能定位 - 在屏幕边界自动调整位置
- 🔧 Y 轴精度优化 - 显示 2 位小数
- 🔧 Y 轴范围只基于目标基金数据计算
- 🔧 新添加的基金显示在列表最前面

### 修复
- 🐛 修复基金净值获取问题 - 使用正确的 AKShare API
- 🐛 修复涨幅计算不准确问题 - 改用复权净值
