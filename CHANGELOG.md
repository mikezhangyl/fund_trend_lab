# 更新日志 | Changelog

## [1.1.0] - 2026-01-14

### 新增
- ✨ **复权净值计算** - 考虑分红再投资，准确显示真实收益率
- ✨ **新布局** - 一行 3 个基金的网格布局
- ✨ **时间区间选择器** - 支持 1年/3年/5年 统一切换
- ✨ **基金排序功能** - 按最新/名称/代码排序
- ✨ **基金删除功能** - 支持删除基金及其所有历史数据
- ✨ **固定顶部 Header** - 滚动时始终可见

### 优化
- 🔧 Tooltip 智能定位 - 在屏幕边界自动调整位置

- 🔧 新添加的基金显示在列表最前面

### 修复
- 🐛 修复基金净值获取问题 - 使用正确的 AKShare API
- 🐛 修复涨幅计算不准确问题 - 改用复权净值
