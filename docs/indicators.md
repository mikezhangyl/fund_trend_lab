# 技术指标说明

## 卡片底部指标

### RS (相对强度)
**公式**: 基金涨幅 - 沪深300涨幅

| 数值 | 含义 |
|------|------|
| RS > 0 (绿色) | 跑赢大盘 |
| RS < 0 (红色) | 跑输大盘 |
| RS > 5% | 强势领先 |

**示例**: RS: +3.5% 表示该基金比沪深300多涨了3.5%

---

### 动量
**公式**: (今日净值 - 20日前净值) / 20日前净值 × 100%

| 数值 | 含义 |
|------|------|
| > 0 (绿色) | 近期上涨趋势 |
| < 0 (红色) | 近期下跌趋势 |
| > 10% | 强劲上涨 |

**示例**: 动量: +5.1% 表示近20个交易日上涨了5.1%

---

### 蓄势 (紫色标签)
当**波动率压缩比 < 0.8** 时显示

**含义**: 近期波动明显收窄，价格在窄幅震荡，可能正在"蓄势"，突破后往往有较大行情

---

## 卡片标签

### 🔥 热门 (红色)
**条件**: 综合评分 ≥ 4

表示该基金同时具备多个积极信号：
- 明显跑赢大盘
- 上涨动能强劲
- 可能还有波动率压缩

**建议**: 重点关注

---

### 📈 关注 (橙色)
**条件**: 综合评分 2-3

表示有一定积极信号，但不如"热门"强烈

**建议**: 纳入观察名单

---

## 评分规则

| 条件 | 得分 |
|------|------|
| RS > 5% | +2 |
| RS > 0% | +1 |
| 动量 > 10% | +2 |
| 动量 > 0% | +1 |
| 波动率压缩比 < 0.6 | +2 |
| 波动率压缩比 < 0.8 | +1 |

---

## 使用提示

1. **"热门"不等于"必涨"** - 这些指标只是统计信号，不是预测
2. **结合图表看** - 指标需要配合走势图理解
3. **关注"蓄势"** - 长期横盘后的突破往往比较有力
4. **对比RS** - RS持续为正说明选股能力强于市场
